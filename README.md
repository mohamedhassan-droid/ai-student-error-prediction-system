AI student error predictor system 

Clear description of the project 

This AI tool will analyze how students solve engineering problems, identify patterns of mistakes, and predict future errors. So it will improve student performance through targeted feedback instead of normal study buddies, who provide you with the answer immediately. 
So the user inputs their step-by-step solutions or uploads their work, then the AI reads and evaluates the missing steps, incorrect assumptions, and conceptual misunderstandings. Using all the AI will provide likely mistakes the student could face on future exams. The key features are step-by-step solution analysis, prediction of future mistakes based on the students' patterns, error detection and explanation, and personalized practice problems for the student. 
The impact and innovation is that this AI moves beyond traditional answer generation, it focuses more on improving students' thinking and how they're solving problems rather than just getting the answer. This AI is more towards engineering students.
